## 公寓简介
Equinox公寓包含24小时免费健身房，3个公共客厅，提供电视，学习室，会议室，休闲区，如果有客人拜访或者组织party，可以免费租用公共区域。
公寓楼顶有超大露天阳台，可以烧烤，同时也能远眺西雅图的天际线。可以欣赏日落日出。
公寓还提供Parcel Locker服务，快递无忧，所有到达的快递都会被安在放置在locker room，需要短信验证码才可访问。
